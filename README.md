# Sunrise API

Backend services for the Sunrise Atelier platform.

## Overview

Sunrise API powers the Sunrise Atelier ecosystem, including authentication, contributor management, project tracking, and future platform features.

## Technology Stack

* Next.js
* TypeScript
* React
* Python
* FastAPI
* PostgreSQL
* SQLAlchemy
* Alembic

## Getting Started

### Prerequisites

* Python 3.14+
* PostgreSQL
* Poetry

### Installation

```bash
poetry install
```

### Run the application

```bash
poetry run uvicorn app.main:app --reload
```
